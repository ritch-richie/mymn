## Local development

Requirements: Node.js and npm (or [nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Build

```sh
npm run build
```

## Deploy

Build the project and deploy the `dist` folder to your hosting provider (Vercel, Netlify, GitHub Pages, etc.).
